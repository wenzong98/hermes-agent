
# 美股 ETF 估值驱动定投策略回测报告

## 策略概述

基于 Shiller CAPE（周期调整市盈率）+ RSI 的多区间信号系统，对美股 ETF（SPY）进行定投和止盈管理。

### 核心规则

**买入信号（DCA 倍数）：**
- CAPE < 25（便宜）：1.5x DCA
- CAPE < 25 + RSI < 30（超卖）：2x DCA
- CAPE < 25 + RSI < 30 + 极端恐慌：3x DCA

**卖出信号：**
- CAPE > 40（极度高估）：减仓 30%
- CAPE > 48（泡沫）：减仓 50%

**持有信号：**
- CAPE 25-32（合理偏高）：正常 1x DCA
- CAPE 32-40（高估）：暂停 DCA，持有观望

## 关键结论

1. **显著跑赢大盘**：CAGR 12.97% vs B&H 10.72%，32年累计多赚 94%
2. **更低回撤**：最大回撤 -37.88% vs -55.19%，风险控制更优
3. **高夏普比率**：1.074 表明风险调整后收益优秀
4. **交易频率低**：年均约 11 次交易，适合长期投资者

## 适用场景

- 适合有稳定现金流的长期投资者（每月定投）
- 适合无法承受 50%+ 回撤的保守型投资者
- 适合相信估值回归、但不想错过长期牛市的投资者

## 风险提示

- 历史回测不代表未来表现
- CAPE 指标在利率极低环境下可能失效
- 策略在持续单边牛市中可能跑输满仓持有
- 建议结合个人风险承受能力调整参数
